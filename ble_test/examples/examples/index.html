Evothings Launchpad

This file is meant to be put on a web server, and be opened in a mobile browser. From this page you can launch the Evothings Viewer app, using the custom evo: and evos: protocols (evo: maps to http: and evos: maps to https:). You can use this on your own web pages, or in emails, to share apps with others.

Evothings Viewer must be installed on the mobile device for this to work.

  • Open evo://evothings.com/uploads/demo/demos.html
  • Open evos://evos.com/uploads/demo/demos.html
