This app is written in ECMAScript 6.

When clicking Run in the Workbench, the app is built and ECMAScript 6 code is translated to regular JavaScript code (ECMAScript 5).

The application code is in folder app.

The built code is in folder www.

Settings are in file evothings.json.

Overview of settings:

  • app-dir: Application directory.
  • www-dir: Build output directory.
  • index-file: Main HTML file in application directory.
  • dont-build: Array of subdirectories in the application directory that should not be processed, these directories are copied verbatim to the output directory.
  • app-uuid: UUID for the app.
  • app-icon: App icon file name.
